# 📚 Book Matchmaker: Recommender System

A machine learning-powered web application that recommends books based on content similarity. Built with **Python**, **Scikit-Learn**, and **Streamlit**.

## 🚀 Features
- **Content-Based Filtering**: Uses TF-IDF Vectorization and Cosine Similarity to find books with similar themes and authors.
- **Interactive UI**: Searchable dropdown to pick from 10,000 books.
- **Visual Results**: Displays book covers and titles in a clean, responsive grid.
- **Fast Performance**: Implements Streamlit caching for instant recommendation generation.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Data Library:** Pandas
- **ML Library:** Scikit-Learn (TF-IDF, Linear Kernel)
- **Web Framework:** Streamlit
